Challenge #1

Write code to convert 111011111 (base 2) into base 10

Pseudocode


# 0. Assign n to the number in base 2 to be converted to base 10
#    n = 111011111

# 1. Assume output is n in base 10, and set it to 0
#    output = 0

# 2. Convert n to a string
#    eg: nstr = str(n)

# 3. Figure out how long the string (nstr) and call it nlen

# 4. Loop through the length of the string

# 5. For each iteration of the loop:
#    (we will loop upwards, from 0 till len(nstr)-1)
#    this means that we should expect to read the digits from right to left
# 5a.   Assign the character by the offset indicated by the loop variable (i.e., i)
#       Eg, x = nstr[len(nstr) - 1 - i]
#        (this may take time to understand)
# 5b.   Convert x to an integer, call it y
# 5c.   Multiply and sum it up (as per the algorithm)
#       output += y * pow(2, i)

# 6.    Print output

# 7.    Rejoice!

Answer #1

Write code to convert 111011111 (base 2) into base 10


n = 111011111
output = 0
nstr = str(n)
nlen = len(nstr)

# Our ever erstwhile loop variable
i=0

while (i < nlen):

    x = nstr[nlen - 1 - i]
    y = int(x)
    output += y * pow(2, i)

    i=0

# This should print 479
print output

Challenge #4

Write code to convert 73 (base 10) into binary (base 2)

Pseudocode


# 0. Assign n to the number in base 10 to be converted to base 2
#    n = 73

# 1. Assume output is n in base 2, and set it to an empty string
#    output = ""

# 2. Loop while out repeated division of 73 doesn't go down to 0
# 2a.   Use divmod() to get the quotient and remainder
# 2b.   Prepend the remainder to output

# 3. Convert output to an integer and print it out

# 4. Bazinga!

Answer #4

Write code to convert 73 (base 10) into binary (base 2)


n = 73
output = ""

while n > 0:

    n, remainder = divmod(n, 2)
    output = str(remainder) + output

# This should print 1001001
print int(output)

def dec2bin(n):

    output = ""

    while n > 0:

        n, remainder = divmod(n, 2)

        # Prepend the remainder
        output = str(remainder) + output

    return int(output)

# This should print 1001001
print dec2bin(73)
